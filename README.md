# TripSync – Travel Planning and Group Coordination Platform

TripSync is a web-based travel planning platform built using **HTML, CSS, PHP, and MySQL**. The system allows users to discover destinations, plan trips, and connect with other travelers who are visiting the same place.

Users can register, log in, create trips with details such as destination, travel date, budget, and group size, and view upcoming group trips organized by other users. The platform also enables travelers to join travel groups through **WhatsApp invite links**, making it easier to coordinate and communicate with fellow travelers.

## Key Features

* User registration and login system
* Destination browsing
* Trip planning with date, budget, and group size
* Travel groups listing
* Join travel groups through WhatsApp invite links
* Personal trip dashboard
* Reviews and likes for destinations

## Technologies Used

* **Frontend:** HTML, CSS
* **Backend:** PHP
* **Database:** MySQL
* **Environment:** XAMPP / Apache Server

TripSync aims to simplify travel planning and help users connect with like-minded travelers for a better group travel experience.
